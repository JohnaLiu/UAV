This doc is for using "UAVbehavior_2.0" project in Unity.

1. When you in Game, you can use R/B/Y with ↑↓←→ to control the crowd.
2. With different thresholds of safe, watch, approach, and threaten, the UAV will response with different actions.
3. You can use 1(red camera), 2(yellow camera), 3(blue camera), 4(UAV camera), 5(Hokuyo camera), 8(overhead camera), 9(reverse camera) and 0(main camera) to change the view in the screen.
